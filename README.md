Yet another PDF Reader with respect to your privacy. 
The project based on [AndroidPdfViewer](https://github.com/TalbotGooday/AndroidPdfViewer) fork.

## Features
- No ads
- No tracking
- Open Source
- Day-night mode
- Multiple documents
